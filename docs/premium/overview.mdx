# Overview

[Apollo Premium](https://apollo.fyi/premium) unlocks additional features for **everyone** in a
Discord server.

Premium enhances the vanilla Apollo experience. A membership provides tools to make
managing your community easier while also giving you more options to customize
the experience.

## Features

Some of the features included with Apollo Premium include:

- Custom signup options and templates
- Attendee roles
- Custom event colors
- Restricted events
- Cutoff time for new signups
- Automatic thread management for attendees
- Unlimited recurring events
- Automatic Google Calendar sync
- Daily or weekly event digests
- **And more!**
